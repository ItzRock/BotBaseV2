## Requirements
- git
- [Node.js](https://nodejs.org/)
- [MongoDB server](https://www.mongodb.com/try/download/community)
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ItzRock/Hue-v3.0.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
